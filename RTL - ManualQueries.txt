# Manual Queries

RTL Queries
    + getBy & getAllBy
    + queryBy & queryAllBy
    + findBy & findAllBy

+ We can use the regular querySelector DOM API to find elements
    eg:
        const {container} = render(<MyComponent/>)
        const foo = container.querySelector('[data-foo="bar"]')
